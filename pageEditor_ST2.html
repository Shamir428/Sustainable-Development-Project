These pages were designed to work together to create a complete and structured user experience. The splash screen introduces the website, the simulator provides interaction, and the content page offers supporting information related to health and wellbeing.

Splash Screen

The splash screen was developed using standard HTML elements such as <div>, <h1>, and <p> to structure the content. A container element was used to group all components together, ensuring a consistent layout and alignment.

CSS was used to create a visually appealing full-screen background image using properties such as background-size: cover and background-position: center. An overlay with a semi-transparent background colour was added using rgba values to improve text readability against the background image.

In terms of functionality, a meta refresh tag was used to automatically redirect the user to the home page after four seconds. Additionally, JavaScript was implemented to create a countdown timer and a “Skip Intro” button, allowing users to manually navigate to the next page if desired.

These design and implementation choices were made to provide a smooth and professional introduction to the website while maintaining accessibility and usability.

Action Impact Simulator

The Action Impact Simulator is the main interactive feature of the website and was developed using a combination of HTML, CSS, and JavaScript. The structure of the page uses semantic HTML elements such as <header>, <main>, and <div> to organise content clearly and improve accessibility.

Each action is presented as a card using div elements, which allows content to be grouped logically and styled consistently. CSS Grid was used to create a responsive layout, ensuring that the cards adjust to different screen sizes. Additional styling techniques such as box shadows, hover effects, and border radius were applied to enhance the user interface and provide visual feedback.

JavaScript was used to implement all interactive functionality. Event handling was used to detect when a user selects an action, and the total score is updated dynamically. The use of classList ensures that each action can only be selected once, preventing errors in the score calculation.

The simulator also includes a progress bar that updates based on the user’s score, providing a visual representation of progress. Furthermore, the background image changes dynamically depending on the user’s impact level (low, medium, or high), which enhances user engagement and reinforces feedback visually.

Content Page

The content page was designed to provide informative and relevant information about health and wellbeing. Structured HTML elements such as <section>, <h2>, and <p> were used to organise the content into clear and readable sections.

CSS was used to control layout, spacing, and typography. Consistent margins, font sizes, and colours were applied to ensure readability and visual consistency across the page. The design focuses on clarity, allowing users to easily understand the information presented.

The content was also linked conceptually to the simulator, helping users understand how their choices in the simulator relate to real-life habits.

Accessibility Considerations

Accessibility was an important consideration during development. All images include alternative text (alt attributes) to support screen readers and assist visually impaired users. Semantic HTML elements such as <header>, <nav>, and <main> were used to improve page structure and navigation.

Colour contrast was carefully chosen to ensure that text remains readable against backgrounds, particularly when using images. Interactive elements such as buttons and cards are clearly visible and easy to use, improving usability for all users.

These accessibility features ensure that the website is inclusive and usable by a wide range of users.

Compliance with JANET Regulations

The website complies with JANET regulations by ensuring that all content is appropriate, respectful, and suitable for an academic environment. No harmful, offensive, or misleading content is included.

External resources such as images are used responsibly, and care has been taken to ensure that they are either royalty-free or used appropriately. Academic integrity has been maintained by ensuring that all work reflects personal understanding and effort.

Additionally, the website does not collect or misuse personal data, ensuring that it follows responsible web publishing practices.
