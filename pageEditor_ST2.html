Splash Screen

The splash screen was built using HTML elements such as <div>, <h1>, and <p> to structure the content.

CSS was used to create a full-screen background image using background-size: cover and background-position: center. An overlay was applied to improve text readability.

A meta refresh tag was used to automatically redirect users after 4 seconds. JavaScript was also used to implement a countdown timer and a skip button.

Action Impact Simulator

The simulator uses semantic HTML elements such as <header>, <main>, and <div> to structure the page. Each action is displayed as a card.

CSS Grid was used to arrange the cards responsively. Styling techniques such as hover effects, shadows, and spacing improve usability and visual design.

JavaScript was used to handle interactivity, including updating scores, preventing duplicate selections, and dynamically changing the background based on user impact level.

Content Page
