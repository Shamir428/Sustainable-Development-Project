Splash Screen

The splash screen was built using HTML elements such as div, headings, and paragraphs. CSS was used to create a full-screen background image and overlay for readability.

A meta refresh tag and JavaScript countdown were used to redirect the user and provide interaction.

Action Impact Simulator

The simulator was built using HTML structure with cards representing actions. CSS Grid was used to create a responsive layout and styling such as shadows and hover effects improved usability.

JavaScript was used to handle interaction, update scores, prevent duplicate selections, and dynamically change the background image depending on the user’s impact level.

Content Page

The content page uses structured HTML elements such as sections and headings to organise information clearly. CSS was used to ensure consistent spacing, typography, and readability.

Accessibility Considerations

Accessibility was considered by using alt text for images and semantic HTML elements. Colour contrast and layout ensure readability, and interactive elements are clearly visible.
