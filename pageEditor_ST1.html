Home Page

Link: View Home Page

HTML Structure

The Home Page was built using semantic HTML elements such as <header>, <main>, and <section> to clearly organise content. Each section was structured to separate the introduction, card-based content, and gallery area.

CSS Techniques

Flexbox was used to align the navigation bar and ensure it remains responsive across different screen sizes. A reusable card design was created using the global CSS to maintain consistency across all pages. Hover effects were added to improve interactivity and visual feedback.

Design and Implementation Decisions

A card-based layout was chosen to present information in a clear and structured way. This improves readability and allows users to quickly understand key points. The colour scheme was kept consistent with the global CSS to maintain a unified visual identity across the website.

Gallery Page

Link: View Gallery

HTML Structure

The Gallery Page was structured using a grid layout to display multiple images in an organised and responsive way. Each gallery item contains an image and descriptive text related to the SDG theme.

CSS Techniques

CSS Grid was used to create a flexible and responsive gallery layout. Additional styling such as hover effects and spacing were applied to make the gallery visually engaging and easy to navigate.

JavaScript Functionality

JavaScript was used to implement interactivity. When a user clicks on an image, an extended view (modal) is displayed showing a larger version of the image and additional information. Users can also close this view, improving usability.

Design Decisions

A modal design was used to display extended content without redirecting users to another page. This improves user experience by keeping navigation simple and intuitive.

Challenges and Solutions

One challenge was aligning the navigation bar correctly across all pages. This was resolved by using Flexbox and adjusting spacing within the header.

Another issue was the gallery initially appearing empty or poorly spaced. This was fixed by properly configuring the grid layout and ensuring images had correct sizing and margins.

I also faced difficulties maintaining consistent styling across pages, which was solved by refining the global CSS and reusing shared classes.

Accessibility Considerations

Accessibility was considered throughout development by using semantic HTML elements to improve structure and readability for assistive technologies.

All images include descriptive alt text to support users who rely on screen readers. Additionally, focus styles were implemented to allow keyboard navigation.

Colour contrast was also considered to ensure text is readable against background colours.

Compliance with JANET Regulations

All content included in the website is appropriate and relevant to the chosen SDG. External images were sourced from royalty-free platforms and are properly referenced.

The website avoids plagiarism by ensuring all written content is original. The design and implementation follow good web development practices and maintain a professional standard.
